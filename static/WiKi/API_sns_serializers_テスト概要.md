CustomUserSerializer のテスト

setUp メソッドで CustomUser インスタンスを作成し、シリアライザーに渡す。
テストメソッドでシリアライザーが期待通りに動作しているか検証する。
シリアライザーのデータに期待されるフィールドが含まれていることを確認する。
各フィールドの内容が正しいことを確認する。
